##### Problem-Solving

- 문제를 풀이 후 가장 상단에 다음 주석 추가
  ```text
      # 풀이 시간 : 00분
      # 문제 핵심 내용 : 다음과 같은 원리를...
      # 회고 : 이러한 점에서 다음번에는...
  ```
- Commit convention rule : [문제출처(플랫폼)] 문제번호*문제이름*사용언어 (문제번호는 없을 경우 생략)
  - 플랫폼 네이밍 rule

    - [JOL] - 정올
    - [ETC] - 그외
